# streamlit-application
Streamlit Application


## Environment

```bash
python -m venv venv
source ./venv/Scripts/activate
python -m pip install --upgrade pip
pip install pandas matplotlib seaborn plotly requests streamlit
```

## Run Server

- Local URL: http://localhost:8501

```bash
treamlit run app.py
```
